### Keyup Evt
1.Deep sea
- Render same element with component 컴포넌트로 가오리 랜딩
- Keyup Event 키업 이벤트 : 스타일 변경 (up,down,right,left 위치 조정) 
- DarkMode 다크모드 설정
- Connect Size with Opacity : 컴포넌트에 전달되는 사이즈에 따라 다른 불투명도로 설정 (가오리)
- Sound source : 사용자의 원함에 따라 배경음악 turn on/off
